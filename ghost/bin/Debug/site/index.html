Панель переключения рабочих окон

Компонент, предназначенный для расположения четырех дочерних элементов.

Три элемента (первый, третий и четвертый) предназначены для перехода между рабочими областями программного средства. Оставшийся элемент (второй) предназначен для вывода справочной информации.

Расположение: крайний левый компонент.



Главная

Дочерний элемент панели переключения, предназначен для перехода на главное рабочее окно.

Расположение: первый элемент в "панели переключения", в виде "домика".




Справка

Дочерний элемент панели переключения, предназначен для перехода на справочную страницу.

Расположение: второй элемент в "панели переключения", в виде "знака вопроса вписанного в окружность".




Автоматическая отправка сообщений

Дочерний элемент панели переключения, предназначен для перехода в меню автоматической отправки данных.

Расположение: третий элемент в "панели переключения", в виде "двух закругленных стрелок".

Узнать о возможностях данного рабочего окна можно ТУТ.




Настройки

Дочерний элемент панели переключения, предназначен для перехода в меню настроек.

Расположение: последний (четвертый) элемент в "панели переключения", в виде "шестерни".

Узнать о возможностях данного рабочего окна можно ТУТ.




Панель контактов

Группа из двух тесно связанных элментов, предназаченых для упращения использования древовидной структурой хранения данных.

Расположение: второй слева, после панели переключения.



Панель создания ветвей

Панель, предназначенная для хранения элементов управления. Включает в себя три элемента, для создания новых ветвей и структур в компоненте хронящем "контакты", такие как:
- новый пользователь;
- новая папка;
- новая группа.

Расположение: верхняя часть компонента контакты.



Создание папки

На панели создания ветвей, присутствует элемент создания "папкa". Для создания папки необходимо нажать один раз, левой кнопкой мыши по иконке в виде "папки".

Расположение: второй слева элеменет на панели создания ветвей.

После нажатия на элемент создания папки, на компоненте окно контактов появится элемент "папка".




Создание подключения

На панели создания ветвей, присутствует элемент создания "пользователь". Для создания подключения к пользователю, необходимо нажать один раз, левой кнопкой мыши по иконке в виде "одного человечка".

Расположение: первый слева элемент на панели создания ветвей.

После нажатия на элемент создания "пользователь", появится окно для ввода данных необходимых для подключения к другому пользователю.

Панель создания подключения

В окне присутствует пять элементов, три из них это поле для ввода данных, остальные два - кнопки взаимодействия с окном подключения. Данные которые необходимы для подключения, находятся на рабочей панели "настройки". Для подключения к другому компьютеру или экземпляру программы, используйте данные для подключения того экземпляра программы, к которому вы хотите подключится.

В данном примере будут использоваться данные собственного экземпляра (подключение к себе).

Помните, при не корректно введенных данных или при отсутствии в сети подключаемого экземпляра, программное средство не создаст подключение.

Если все прошло успешно, на компоненте окно контактов появится элемент "пользователь".




Создание группы

На панели создания ветвей, присутствует элемент создания "группа". Для создания множественного подключения к пользователям. Чтобы создать группу, необходимо нажать один раз, левой кнопкой мыши по иконке в виде "трех человек".

Расположение: третий слева элемент на панели создания ветвей.

После нажатия на элемент создания группы, на компоненте окно контактов появится элемент "группа".




Управление группой

После создания элемента группа, необходимо создать элемент пользователь или использовать уже существующее подключение.

Для того чтобы добавить пользователя в вашу новую группу, необходимо навести указатель мыши на нужного вам пользователя, существующего на панели контактов, зажать левую кнопку мыши и перетащить элемент пользователь на элемент группа.

После отправки первого сообщения, если получатель его получит, у пользователей которых в добавили в группу, автоматически создастся группа и одновременно подключение.




Окно контактов

Компонент программного средства предназначенный для хранения структур, таких как:
- папка;
- пользователь;
- контакт;
- группа.

Расположение: элемент ниже, панели создания ветвей.



Папка

Дочерний элемент "окно контактов", предназначенный только для хранения других элментов.

Чтобы открыть контекстное меню элемента "папка", необходимо навести указатель мыши на нужный вам элемент "папка" и нажать правую кнопку мыши.

Вложить папку в другую папку можно двумя способами: Первый способ это нажать на элемент "папка" правой кнопкой мыши, чтобы вызвать контекстное меню. В контекстном меню необходимо выбрать пункт "Добавить папку".

Второй способ, это создать "новую папку", затем перетащить новую папку на элемент "папка" в которую вы хотите вложить новую папку.

Чтобы изменить название папки, необходимо нажать на элемент "папка" правой кнопкой мыши, чтобы вызвать контекстное меню. В контекстном меню необходимо выбрать пункт "Изменить".

Далее появится окно редактирования папки.

В окне присутствует три элемента, один из них это поле для ввода данных, остальные два - кнопки взаимодействия с окном редактирования. В поле для ввода данных присутствует текущее название папки.

Чтобы изменить название, измените текст в поле для ввода текста и нажмите на кнопку "изменить". После нажатия на кнопку "изменить", название папки изменится, окно редактирования закроется.

Чтобы удалить папку, необходимо нажать на элемент "папка" правой кнопкой мыши, чтобы вызвать контекстное меню. В контекстном меню необходимо выбрать пункт "Удалить".

После нажатия на пункт "удалить", папка и все ее содержимое будет удалено.

Чтобы удалить содержимое папки, необходимо нажать на элемент "папка" правой кнопкой мыши, чтобы вызвать контекстное меню. В контекстном меню необходимо выбрать пункт "Удалить содержимое".

После нажатия на пункт "Удалить содержимое", все содержимое папки будет удалено.




Пользователь

Дочерний элемент "окно контактов", предназначенный для хранения переписки с пользователем и для хранения списка адресов пользователя.

Чтобы открыть контекстное меню элемента "пользователь", необходимо навести указатель мыши на нужный вам элемент "пользователь" и нажать правую кнопку мыши.

Чтобы перейти к переписке, необходимо нажать на элемент "пользователь" правой кнопкой мыши, чтобы вызвать контекстное меню. В контекстном меню необходимо выбрать пункт "Перейти в диалог".

Чтобы изменить имя пользователя, необходимо нажать на элемент "пользователь" правой кнопкой мыши, чтобы вызвать контекстное меню. В контекстном меню необходимо выбрать пункт "Изменить".

Далее появится окно редактирования имени пользователя.

В окне присутствует пять элементов, три из них это поля для ввода данных, остальные два - кнопки взаимодействия с окном редактирования. Первое поля для ввода - текущие, оторбажаемое, имя пользователя. Второе поле для ввода (неизменяемое) - это глобальное имя пользователя во всей сети. Третье поле это открытый RSA ключ пользователя (неизменяемое). В первом поле для ввода данных присутствует текущее название пользователя. Чтобы изменить название, измените текст в поле для ввода текста и нажмите на кнопку "изменить". После нажатия на кнопку "изменить", имя пользователя изменится, окно редактирования закроется.

Чтобы удалить пользователя, необходимо нажать на элемент "пользователь" правой кнопкой мыши, чтобы вызвать контекстное меню. В контекстном меню необходимо выбрать пункт "Удалить".

После нажатия на пункт "удалить", пользователь будет удален и все ее содержимое будет удалено.

Примечание. Если удалить пользователя которому присвоено имя, при новом создание данного пользователя, имя данное вами будет автоматически восстановлено.

Чтобы удалить все адреса пользователя, необходимо нажать на элемент "пользователь" правой кнопкой мыши, чтобы вызвать контекстное меню. В контекстном меню необходимо выбрать пункт "Удалить содержимое".

После нажатия на пункт "Удалить содержимое", все адреса пользователя будут удалены.




Адрес

Дочерний элемент "окно контактов", основная функция это хранение адреса пользователя для прямого подключения.

Чтобы открыть контекстное меню элемента "адрес", необходимо навести указатель мыши на нужный вам элемент "адрес" и нажать правую кнопку мыши.

Чтобы изменить название элемента "адрес", необходимо нажать на элемент "адрес" правой кнопкой мыши, чтобы вызвать контекстное меню. В контекстном меню необходимо выбрать пункт "Изменить".

Далее появится окно редактирования элемента "адрес".

В окне присутствует четрые элементов, два из них это поля для ввода данных, остальные два - кнопки взаимодействия с окном редактирования. Первое поля для ввода - текущие, оторбажаемое, название элемента "адрес". Второе поле для ввода (не изменяемое) - это адрес (IP + PORT) пользователя. В первом поле для ввода данных присутствует текущее название элемента "адрес". Чтобы изменить название, измените текст в поле для ввода текста и нажмите на кнопку "изменить". После нажатия на кнопку "изменить", название элемента изменится, окно редактирования закроется.

Чтобы удалить адрес, необходимо нажать на элемент "адрес" правой кнопкой мыши, чтобы вызвать контекстное меню. В контекстном меню необходимо выбрать пункт "Удалить".

После нажатия на пункт "удалить", адрес будет удален.

Чтобы отправить сообщение пользователю, сначала надо создать подключение и установить соединение. Чтобы установить соединение с пользователем, необходимо выполнить двойное нажатие на элемент "адрес", находящийся в элементе "пользователь". Если подключение будет удачным, поле для ввода сообщения станет активным, в противном случае в поле для ввода текста будет предупреждение о том что пользователь недоступен.

Чтобы отправить подписанное сообщение пользователю, сначала надо создать подключение и установить соединение. Чтобы установить соединение с пользователем, необходимо выполнить двойное нажатие на элемент "адрес", находящийся в элементе "пользователь". Если подключение будет удачным, поле для ввода сообщения станет активным, в противном случае в поле для ввода текста будет предупреждение о том что пользователь недоступен.

Если соединение произошло успешно, можно отправить подписанное сообщение.

Для этого необходимо набрать сообщение в поле для ввода сообщений и нажать на элемент в виде "карандаша", справа от поля для ввода текста.

После нажатия элемента для подписи сообщения, элемент поменяет свой цвет с белого на голубой. Чтобы отключить режим подписи сообщения, необходимо повторно нажать на элемент, после чего он сменит цвет с голубого на белый.

Чтобы отправить подписанное сообщение надо нажать клавишу "Enter".

Подписанные сообщения выделяется среди обычных, текст имеет голубой цвет.

Примечание. В данном примере, подключение и отправка сообщения происходит в одном экземпляре программы.

Чтобы отправить файл пользователю, сначала надо создать подключение и установить соединение. Чтобы установить соединение с пользователем, необходимо выполнить двойное нажатие на элемент "адрес", находящийся в элементе "пользователь". Если подключение будет удачным, поле для ввода сообщения станет активным.

Если соединение произошло успешно, можно отправить файл. Для этого необходимо нажать на элемент в виде "файла", справа от поля для ввода текста, второй слева.

После нажатия элемента для отправки сообщения, откроется проводник в котором нужно выбрать один файл. После выбора файла, в поле для ввода текста будет отображатся процесс отправки данных.

После отправки файла, будет отправлено уведомление о получении файла.

Пользователь который получает файл, может узнать об этом через мониторинг входящих пакетов, находящийся в левом верхнем углу.

Полученые файлы можно найти в корневой папке "./files".




Группа

Дочерний элемент "окна контактов", предназначенный для создания групповых сессий.

Чтобы открыть контекстное меню элемента "группа", необходимо навести указатель мыши на нужный вам элемент "группы" и нажать правую кнопку мыши.

Чтобы перейти к переписке, необходимо нажать на элемент "группа" правой кнопкой мыши, чтобы вызвать контекстное меню. В контекстном меню необходимо выбрать пункт "Перейти в диалог".

После нажатия на пункт "Перейти в диалог", элемент информирования, в "Главном окне", о текущем подключении изменится на название группы.

Чтобы изменить название группы, необходимо нажать на элемент "группа" правой кнопкой мыши, чтобы вызвать контекстное меню. В контекстном меню необходимо выбрать пункт "Изменить".

Далее появится окно редактирования группы. Окно редактирования показано на нижеприведенном изображении.

В окне присутствует четыре элементов, два из них это поля для ввода данных, остальные два - кнопки взаимодействия с окном редактирования. Первое поля для ввода - текущие, оторбажаемое, имя пользователя. Второе поле для ввода (не изменяемое) - это глобальное имя группы во всей сети. В первом поле для ввода данных присутствует текущее название пользователя. Чтобы изменить название, измените текст в поле для ввода текста и нажмите на кнопку "изменить". После нажатия на кнопку "изменить", имя пользователя изменится, окно редактирования закроется.

Чтобы удалить группу, необходимо нажать на элемент "группа" правой кнопкой мыши, чтобы вызвать контекстное меню. В контекстном меню необходимо выбрать пункт "Удалить".

После нажатия на пункт "удалить", группа будет удалена и все ее содержимое будет удалено.

Чтобы удалить всех пользователей группы, необходимо нажать на элемент "группа" правой кнопкой мыши, чтобы вызвать контекстное меню. В контекстном меню необходимо выбрать пункт "Удалить содержимое".

После нажатия на пункт "Удалить содержимое", все пользователи будут удалены из группы.

Для отправки сообщений необходимо перейти в "переписку". Далее ввести сообщение в поле для отправки сообщений и нажать клавишу "Enter".




Автоотправка сообщений

Что такое автоматическая отправка сообщений? Это инструмент позволяющий отправлять сообщения пользователям, которые в данный момент не в сети. Данный инструмент пробует отправлять сообщения к другому пользователю до тех пор пока сообщение не будет отправлено. Каждые 15 секунд срабатывает алгоритм, который проходит по очереди через весь список добавленных сообщений в очередь, и пытается их отправить получателю.

Для начала необходимо перейти на форму автоотправки сообщений. Далее необходимо одиночным кликом левой кнопки мыши выбрать элемент адрес получателя.

Если "адрес" отобразился в строке состояния, значит все сделано правильно.

Далее необходимо ввести сообщение в поле для ввода сообщений. Далее надо нажать клавишу "Enter" чтобы разместить сообщение в очередь на автоотправку.




Окно настройки

Рабочее окно Настройки, предназначено для хранения данных служащих для подключения других пользователей.

Далее необходимо ввести сообщение в поле для ввода сообщений. Далее надо нажать клавишу "Enter" чтобы разместить сообщение в очередь на автоотправку.
